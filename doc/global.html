multiMatchAndFilter(body)

This method handles multiple query parameters with term and range filter combined. It is not necessary to provide all the data below, however it is necessary to send all the keys (like index, type, match, filter etc). 
    http://localhost:3000/api/elastic/search/match
    {
       "index": "shakespeare",
       "type": "line",
       "page": "1",
       "size": "20",
       "match": [{
           "text_entry": "pants"
       },
       {
           "text_entry": "ride"
       }],
       "filter": [{
           "speech_number": "1"
       },
       {
           "play_name": "Antony and Cleopatra"
       }],
       "filterRange": {
           "line_id": {
               "lte": "22300",
               "gte": "21000"
           }
       }
    }
Or minimally, you can call this api like this: 
    http://localhost:3000/api/elastic/search/match
    {
       "index": "shakespeare",
       "type": "line",
       "page": "1",
       "size": "20",
       "match": [{
           "text_entry": "pants"
       }],
       "filter": [],
       "filterRange": {}
    }
